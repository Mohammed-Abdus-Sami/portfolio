# Mohammed-Abdus-Sami — Developer Portfolio

A modern, responsive developer portfolio website built with vanilla HTML, CSS, and JavaScript.

## 🎯 About

This portfolio showcases my full-stack development projects, skills, and experience. It's designed to be fast, clean, and professional — suitable for sharing with potential employers and clients.

## ✨ Features

- **Responsive Design** — Looks great on desktop, tablet, and mobile
- **Smooth Animations** — Scroll reveal effects and gradient accents
- **Dark Theme** — Modern dark UI with gradient highlights
- **No Dependencies** — Pure HTML/CSS/JS, no build step needed
- **Fast Loading** — Lightweight and optimized

## 📁 Structure

```
portfolio/
├── public/
│   ├── index.html    # Main page with all sections
│   ├── styles.css     # All styling (dark theme)
│   └── app.js        # Scroll effects, nav, animations
├── README.md
└── render.yaml       # Render.com deployment config
```

## 🚀 Local Development

Just open `public/index.html` in a browser, or serve it:

```bash
npx serve public
```

## 📦 Deploy on Render

This repo includes a `render.yaml` for one-click deployment on Render.com as a static site.

1. Fork/clone this repo
2. Go to [render.com](https://render.com) → New → Static Site
3. Connect the repo
4. Build Command: (leave empty)
5. Publish Directory: `public`
6. Deploy!
